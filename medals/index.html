Medals

General Medal Rules

  • MG-Team Headless only. (*Disabled for testing)
  • MegaPack only.
  • Only coop, (Humans vs CPUs).
  • Number of human players should be equal or bigger than CPUs.
  • All humans must win.
  • No human should leave the game (quit or timeout) before game ends.
  • All CPUs should be Ultra or Mega and Resource Multiplier > 1.4.

812 games have met the given criteria.
* * Yes, I know , this rules out many games :s, but its an anti-cheat measure. Also Resource Multiplier > 1.4 and CPU Mega and Ultra removes many games, but only these games are worth a medal.

Specific Medal Rules

  • GOLD: All CPUs Mega and Resource Multiplier should be 4.9
  • SILVER: All CPUs Resource Multiplier should be between 2.9 and 4.9
  • BRONZE: All CPUs Resource Multiplier should be between 1.4 and 3
